# ☕ JavaFX Projects

¡Bienvenido a este repositorio! Este espacio está diseñado como una guía práctica y educativa para comprender los **fundamentos** y las **buenas prácticas** en el desarrollo de aplicaciones de escritorio utilizando **JavaFX**.

La finalidad principal es construir una base sólida de conocimiento y, de manera iterativa, ir trabajando en **mejorar nuestras GUIs** (Interfaces Gráficas de Usuario) hacia diseños más modernos, fluidos y profesionales.

---

## 🎯 Objetivos del Repositorio

*   **Fundamentos Sólidos**: Comprender el ciclo de vida de una aplicación JavaFX, el manejo del Escenario (*Stage*) y la Escena (*Scene*).
*   **Buenas Prácticas**: Implementar arquitecturas limpias y separar la lógica de negocio de la vista (patrón MVC/FXML).
*   **Evolución de GUI**: Mejorar continuamente el aspecto visual utilizando hojas de estilo **CSS**, componentes personalizados y maquetación responsiva.
*   **Gestión de Diseños**: Aprender a estructurar interfaces usando contenedores (*Layouts*) como `GridPane`, `VBox`, `HBox` y `BorderPane`.

---

## 🚀 Hoja de Ruta (Roadmap) de Aprendizaje

*   [x] **Fase 1**: Configuración del entorno base de desarrollo.
*   [ ] **Fase 2**: Estructura básica y manejo de eventos (*Event Handling*).
*   [ ] **Fase 3**: Diseño de interfaces visuales con archivos **FXML** y Scene Builder.
*   [ ] **Fase 4**: Estilización avanzada con **CSS** (Temas oscuros y modernos).
*   [ ] **Fase 5**: Optimización de interfaces concurrentes (Hilos en segundo plano).

---

## 🛠️ Tecnologías y Herramientas

*   **Lenguaje:** Java 11+ / 17+ ☕
*   **Framework GUI:** JavaFX (OpenJFX) 🖼️
*   **Gestor de Proyectos:** Maven / Gradle 📦

## 💻 Referencia de Inicialización (Git)

Si deseas clonar, inicializar o replicar la configuración base de este repositorio localmente, puedes guiarte con los siguientes comandos estándar:

```bash
# Inicializar el repositorio local
git init

# Agregar el archivo de documentación principal
git add README.md

# Realizar el primer commit de estructura
git commit -m "first commit"

# Renombrar la rama principal a main
git branch -M main

# Vincular con el servidor remoto en GitHub
git remote add origin https://github.com/lalobarri/javafx_projects.git

# Subir los cambios por primera vez
git push -u origin main
```
